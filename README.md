#  Cricket Highlights Generator

##  Project Description
This project generates cricket match highlights automatically.

It selects important moments like:
- Wickets
- Sixes and Fours
- Half-centuries and Centuries

It ensures:
- Only top 20 highlights
- Full match coverage
- IPL-style output

##  Technologies Used
- C++
- STL (vector, map, sort)

##  How to Run

1. Compile:
   g++ main.cpp -o main

2. Run:
   ./main
